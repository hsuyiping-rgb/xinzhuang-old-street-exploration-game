# 新莊老街探索遊戲

以國小中高年級戶外教育為情境的新莊老街行動闖關遊戲。學生以文化守護者小隊身分，走訪廟宇、老店與歷史巷弄，完成觀察、問答及互動任務。

線上版本：<https://hsuyiping-rgb.github.io/xinzhuang-old-street-exploration-game/>

## 開始使用

環境需求：Node.js 18+。專案不需要安裝外部套件。

```powershell
npm.cmd run dev
```

開啟 `http://127.0.0.1:5173/`。

## 驗證與建置

```powershell
npm.cmd run check
npm.cmd run build
npm.cmd run preview
```

- `check`：檢查 JavaScript 語法並完成正式建置。
- `build`：輸出可部署內容至 `dist/`。
- `preview`：在 `http://127.0.0.1:4173/` 預覽正式建置結果。

## 目前定位

目前版本是可在手機或平板瀏覽器執行的前端原型，包含角色選擇、地圖、GPS 模擬、關卡互動、背包、計分與成果報告。每個關卡另有一項必做的田野證據任務，依內容要求現場拍照、口述錄音或至少 20 字的文字紀錄；照片會壓縮後納入成果牆。

進入景點關卡後，可先播放搭配情境照片的中文語音導覽。作答回饋會依各景點與題目線索補充文化意義或提供重新觀察方向；單純移動與解鎖景點不播放掌聲或鼓勵語音。

第一次使用錄音功能時，瀏覽器會要求麥克風權限。若裝置無法直接錄音，可改用關卡內的錄音檔上傳功能。實地 GPS、真正的 AR 影像辨識及後端資料同步仍屬後續擴充範圍。
